Enhanced Test Form

How to test the Enhanced Form AutoFill Extension:

  1. Install the Chrome Extension in developer mode
  2. Load the extension from the dist folder
  3. Click the extension icon in your browser toolbar
  4. Fill in your form data and click "Save"
  5. Click the "Fill Now" button
  6. Watch all form fields get auto-filled with your data!

Basic Information

Field: name
Field: email
Field: phone
Field: address

Additional Information

Field: website
Field: birthdate
Field: meetingTime

Dropdown Selections

Field: country
Field: experience

Radio Button Selections

Field: employmentType

Checkbox Selections

Field: skills

File Upload

Field: resume (will show visual alert)
Field: coverLetter (will show visual alert)
